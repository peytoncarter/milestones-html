The relationship between the opening and closing tags for the p tag, is to signify when to start the paragraph, and when to end it.
The relationship between the opening and closing tags for the a tag, is to show where to start the reference to the link and where the link, and any comment attached to it, will end.

The attributes on the a tag are: href, id, class, and target. These are their values and its effect:

  • href: "https://google.com"
    -This is used to say that the destinaton of the link is google
  • id: "the-best-link"
    -This is used to point just out one thing in the code that can be changed
  • class: "banana"
    -This is used to pick out multiple things within the code to be changed
  • target: "_blank"
    -This is used to say where the link will open up at. In this case it will open in a new tab
